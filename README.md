# 🚀 .NET Developer Roadmap — 3 Months

> **Goal**
>
> Build job-ready Junior .NET Developers through two complete real-world projects while introducing the architectural patterns, design principles, and development practices commonly used in modern .NET applications.

---

Before joining this roadmap, trainees should already be comfortable with the following topics:

| Topic | Level |
|--------|-------|
| C# | ✅ Solid |
| Object-Oriented Programming (OOP) | ✅ Solid |
| LINQ | ✅ Solid |

| Entity Framework Core | ✅ Intermediate |
| ASP.NET Core MVC | ✅ Basic |
| ASP.NET Core Web API | ✅ Basic |
| Authentication Fundamentals | ✅ Basic |

# 🏗️ Project 1 — E-Commerce MVC

During the first six weeks, trainees will build a complete ASP.NET Core MVC E-Commerce application while learning how enterprise MVC applications are structured.

The project focuses on:

- Layered (N-Tier) Architecture
- Repository Pattern & Unit of Work
- ASP.NET Core Identity
- Session & Caching
- File Management
- Entity Framework Core
- Orders & Checkout
- Transactional Emails
- Product Reviews

---

# 🚀 Project 2 — Event Management REST API

During the final six weeks, trainees will build a production-style REST API using modern backend architecture and patterns.

The project focuses on:

- Clean Architecture
- CQRS & MediatR
- JWT Authentication
- Result Pattern
- Global Exception Handling
- Stripe Payments
- SignalR Notifications
- Reporting & Analytics
- FluentValidation
- Deployment

---

# 🎯 Learning Outcomes

By the end of this roadmap, trainees will be able to:

- Design applications using both N-Tier and Clean Architecture.
- Build secure applications using ASP.NET Core Identity and JWT.
- Apply Repository Pattern and Unit of Work appropriately.
- Manage files, sessions, and caching effectively.
- Integrate external services such as MailKit and Stripe.
- Build REST APIs using CQRS and MediatR.
- Implement structured logging and global exception handling.
- Deploy .NET applications to production environments.
